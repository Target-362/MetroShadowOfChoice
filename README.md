# Метро: Тени выбора

Текстовый квест в мире Метро 2033 / Metro Last Light  
Только твои решения. Без подсказок. Без оценок. Без возврата.

## Описание

Ты — Артём с ВДНХ.  
Тебе предстоит пройти через туннели, станции, поверхность и вернуться назад — если хватит патронов, фильтров и удачи.

- 17 частей
- 7 глав
- 5+ разных концовок  
- Система союзников, артефакт «Сердце Тьмы», здоровье, патроны, фильтры  
- Нелинейные решения с последствиями

## Управление

- Вводи только число (1, 2, 3…)
- Enter — подтверждение выбора
- Игра завершается автоматически при смерти или после финальной сцены

## Особенности

- Чистый Python без внешних зависимостей
- Всё в одном файле (~300 строк логики)
- Легко добавлять новые главы / концовки / предметы
- Система состояний через объект Player
- Универсальная функция выбора show_choice()

## Доступные концовки (спойлер-фильтр)

- Странный свет в темноте
- Цена надежды
- Последний патруль
- Тишина
- Один в темноте
- Неизвестность
(и ещё несколько вариаций в зависимости от комбинации параметров)
